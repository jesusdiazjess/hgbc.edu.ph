Website Transition to PHP Platform Now Underway

Published June 8, 2025


HGBaquiran College, Inc. is excited to unveil its upcoming website transformation, beginning the transition from a static platform to a dynamic, PHP-powered system.

What's Happening?
The institution is currently developing and preparing to launch a new and improved version of this website. This upgrade will include features such as dynamic content, user logins, database integration, and interactive tools to enhance functionality.

What About the Current Site?
The existing website, hosted on GitHub at the URL https://jesusdiazjess.github.io/hgbc.edu.ph/, will remain accessible for a temporary period during the migration process. However, upon successful completion of the transition to the new hosting platform, the GitHub Pages site will be permanently decommissioned and taken offline.
